# Rico MD — 设计规范

> 本文件定义 rico-md 的视觉风格、排版规范和交互模式。AI 修改 UI 时应遵循本文档。

## 视觉风格

极简 + 功能导向。编辑器工具的视觉服务于写作效率，不喧宾夺主。

## 排版规范

### 编辑器区域

- 字体：系统等宽字体
- 字号：14px
- 行高：1.7
- 内边距：16px

### 预览区域

- 字体：正文跟随主题定义，代码用等宽字体
- 正文行高：1.8
- 标题间距：上方 24px，下方 12px
- 段落间距：12px

## 组件规范

### 按钮

- 主要按钮：实心背景 `#2d6dc3`，白色文字，圆角 6px
- 次要按钮：透明背景，边框 `#dfe4ed`，文字 `#3f4a5a`
- 悬停：背景色加深，过渡时间 150ms
- 禁用：opacity 0.5，cursor not-allowed

### 面板

- 顶部导航栏：固定定位，高度 48px
- 侧栏面板：右侧滑出，宽度 320px，背景白色
- 面板切换：过渡时间 200ms，使用 transform 而非 display

### Toast 提示

- 位置：顶部居中，距离顶部 16px
- 尺寸：最小宽度 200px，最大宽度 400px
- 动画：从上方滑入（150ms），3 秒后自动消失
- 类型：成功（绿色）、错误（红色）、信息（蓝色）

### 文档列表

- 每项高度 36px
- 活跃文档：背景高亮 `#edf1f8`
- 悬停：背景 `#f7f9fc`
- 删除确认弹窗居中显示

## 交互规范

### 主题切换

- 点击主题卡片即时生效
- 无需确认、无需刷新
- 切换时预览区域整体重渲染

### 自动保存

- 输入后 5 秒防抖触发
- 状态栏实时显示：`保存中...` → `已保存` / `保存失败`
- 保留 Ctrl/Cmd + S 手动保存

### 文档切换

- 点击左侧列表即时切换
- 当前文档内容自动保存后再加载新文档
- 切换过渡无闪烁

## 响应式策略

| 断点 | 布局 |
| --- | --- |
| > 1024px | 左右分栏（编辑器 + 预览） |
| 769px - 1024px | 左右分栏，缩小间距 |
| ≤ 768px | 上下布局（编辑器 40% + 预览 60%） |

移动端额外调整：
- 隐藏部分工具栏按钮
- 面板改为全屏覆盖
- 按钮全宽显示
